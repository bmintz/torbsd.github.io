tor(1)

    DNSPort [address:]port|auto [isolation flags]

	Se for diferente de zero, abra esta porta para ouvir os pedidos de DNS via UDP e resolva-os anonimamente.
	Válido apenas para registros dos tipos A, AAAA e PTR - não lida com tipos de solicitação de DNS arbitrárias.
	Caso a porta esteja definida como "auto", tor(1) escolherá uma porta para você (automaticamente). Esta 
		opcão pode ser utilizada mais de uma vez, para disponibilizar o serviço em diferentes portes e/ou 
		endereços IP. (Padrão: 0).
